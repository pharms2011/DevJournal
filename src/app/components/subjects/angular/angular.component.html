Angular

What is Angular?

Angular is a modern TyppeScipt framework, which is focused on creating reactive single page applications.

What is Typscript

Typescript is a superset of Javascript that is much more similar to Java or any other object oriented language. Type script is transpiled when the angular app is built into vanilla javascript which runs much more easily in the browser.

Components

Angular Components are standalong page elements that make up our angular application. When generating a Component, Angular will generate the Typscript, HTML, and CSS files as well as a file used for testing. The main core of Angualr is the modularity of Components that help to divide complexity down and to reuse code as much as possible.
