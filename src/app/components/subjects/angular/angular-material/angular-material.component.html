Angular Material

What is Angular Material?

Angular Material is a prebuilt framework for angular components made by the angular team. The components are made to be responsive and to run well on mobil and desktop environments. Material itself was designed by google to sythisis design principles as well as clas platform unified look and feel.

Principles

1: Material is a metaphor

Material design handles elements as if they are 3 dimensions, just like material in the real world. This means using things like shadow to give this illusion.

2: Bold,graphic, and Intentional

Color pallets and typography is chosen intentionally to create an environment. 3: Motion provides meaning

Animations in material are used to convey what is happening on the page and in the application. this is used rather than jarring motions.

Angular Flex Layout Box Model

A Flex container is a box created by an element which has a computed dispaly of flex or inline flex. Flex items are elements within this box.
