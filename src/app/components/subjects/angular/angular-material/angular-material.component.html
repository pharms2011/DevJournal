Angular Material

What is Angular Material?

Angular Material is a prebuilt framework for angular components made by the angular team. The components are made to be responsive and to run well on mobil and desktop environments. Material itself was designed by google to synthesis design principles as well as class platform unified look and feel.

Principles

1: Material is a metaphor

Material design handles elements as if they are 3 dimensions, just like material in the real world. This means using things like shadow to give this illusion.

2: Bold,graphic, and Intentional

Color pallets and typography is chosen intentionally to create an environment. 3: Motion provides meaning

Animations in material are used to convey what is happening on the page and in the application. this is used rather than jarring motions.

Angular Flex Layout Box Model

A Flex container is a box created by an element which has a computed display of flex or inline flex. Flex items are elements within this box. Flex box is fully supported in Angular. There's nothing special about it as far as I can tell for now.

Angular Custom Component Styling

If there is one thing I hate about Angular Material, it's how hard it can be to give custom styling to provided components. The only way I have been able to manipulate simple CSS features on these components is to describe where they will be displayed in gratuitous detail in the style.css file, like below.

Ex:

.mat-expansion-panel-content .mat-expansion-panel-body .mat-nav-list{{'{'}}

padding: 0;

{{'}'}}
